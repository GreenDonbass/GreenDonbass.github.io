О проекте

Целью данного проекта является достижение большей осведомлённости о проблемах экологии и путях их решения среди жителей Донбасса.

Задачи проекта

Создание сайта

Разработать сайт для крупного экологического общественного движения ДНР «Зелёный Донбасс», чья информационная поддержка на данный момент основана только на группе в социальной сети «ВКонтакте».
